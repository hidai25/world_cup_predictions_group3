2018, the dataset has 3564 rows and 11 columns. This is the clean dataframe used for our analysis and model construction:

    if 'M' in strVal:
        return int(float(strVal.replace('M', '')) * 1000000)
    elif 'K' in strVal:
        return int(float(strVal.replace('K', '')) * 1000)
